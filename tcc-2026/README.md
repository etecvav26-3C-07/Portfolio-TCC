# Informações sobre o projeto
## Breve resumo
Será feito um site para facilitar o acesso a materiais relacionados a renderização 3D, onde iremos abordar vários temas, um deles será sobre o programa Blender, uma ferramenta muito boa para animações e modelagem 3D onde tudo é renderizado por dentro dele, porém por ter muitas ferramentas o Blender tem uma curva de aprendizado muito difícil para alguns.
## Pontos principais
- Indexação de conteúdos sobre renderização 3D
- Auxilio para quem quer começar
- Facilitação de encontrar  materiais de estudo
- Site simples e prático
